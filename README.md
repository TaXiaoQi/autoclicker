适用于 Fabric 1.21.10 版本的自动攻击模组。

主要用途：挂经验塔，半自动猪人塔、刷怪塔；

使用方法：安装后，对准适用目标后按下绑定按键（默认 F8），将自动攻击；

适用目标：盔甲架、敌对生物；

效率：每 20 Ticks 进行一次攻击；

实现方法：基于攻击事件实现，非鼠标模拟，不会抢鼠标；

优点：

1、支持全自动，开启后可以将游戏窗口最小化挂载在后台 ，不受GUI影响

2、支持特定生物，可以一直开着，不影响放置方块、玩家交互、手动攻击等等 全自动打怪就算了

3、支持更改启动按键，在 游戏设置-按键控制-按键绑定-杂项 中可找到“自动攻击按键”进行更改。

缺点：

由于是走事件触发且频率固定，会被某些反作弊捕捉并判定为异常玩家进行处理，请在使用前询问服务器规则，避免被封号
