适用于 Fabric 1.21.10 版本的自动攻击|自动放置模组。

主要用途：

生电前期用来挂机各种半自动设备（经验塔，半自动猪人塔、刷怪塔、半自动树场）等，支持自动种植和催熟



使用方法：

安装后，对准适用目标后按下绑定按键（默认自动攻击是F8、自动种植的F9），将全自动进行攻击、种植；



适用目标：

自动攻击适用于盔甲架、敌对生物（可在模组设置中选择中立生物和被动生物）

自动种植适用于各类主世界、下界的植物、树苗，支持骨粉催熟；你可以 左骨粉 右植物 直接被动的全自动种植



效率：

自动攻击默认是每 20 Ticks 进行一次攻击，支持在模组设置中自定义

自动种植默认是每5tick进行一次种植或催熟，支持在模组设置中自定义；



优点：

事件实现，模组并非传统鼠标模拟，采用事件实现效果，不会抢鼠标，不受游戏的GUI 影响

全自动托管，开启后可以将游戏窗口最小化挂载在后台 ，不耽误使用电脑

无感游玩，设定了生物、方块、物品范围，防呆纯萌新向，不影响玩家手动操作或者游玩

自定义丰富，在模组设置中可自由设置效率、类型，支持添加随机刻和随机跳过，避免被部分服务器反作弊抓捕

兼容度高，模组并未修改游戏本体，理论上兼容所有模组，如遇不兼容的mod，请与我联系 QQ群 172442817



缺点：

由于本模组属于辅助类全自动模组，虽然有随机刻和随机跳过，但是依旧有概率被部分服务器反作弊识别，请各位玩家在使用前先与腐竹进行确认，避免被封禁
